# Zig Pong
Pong made in Zig + Raylib

![pong demo](./game_demo.png)

- Zig: 0.14
- Raylib: 5.5
- raylib-zig: *Fresh*

# Build and Run
Clone and in source directory run:
```sh
zig build run
```
